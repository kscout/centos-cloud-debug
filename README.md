# Centos Cloud Debug
Centos container image with cloud debug tools installs.

# Table Of Contents
- [Overview](#overview)
- [Develop](#develop)

# Overview
Centos container with:

- Editors:
  - emacs
  - vim
- Tools:
  - git
  - zsh
  - wget
  - bind-utils
  - zsh
  - oc
  - kubectl

# Develop
Build container:

```
make build
```

Run container:

```
make run
```
